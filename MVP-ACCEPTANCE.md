# MVP 手动验收清单

在发布或交付 MVP 前，请按以下清单逐项验证。建议在桌面浏览器与手机浏览器各完成一遍核心流程。

## 环境准备

- [ ] `pnpm lint` 无报错
- [ ] `pnpm build` 成功
- [ ] `docker compose build` 成功（需本机安装 Docker）
- [ ] 本地 `pnpm dev` 可访问 http://localhost:3000
- [ ] 已配置 `.env` 且 `JWT_SECRET` 非默认值

## 认证

- [ ] 未登录访问 `/today` 会跳转登录页
- [ ] 使用 seed / create-user 账号可登录
- [ ] 登录后可访问主界面，登出后需重新登录
- [ ] （可选）配置 `PASSWORD_RESET_SECRET` 后，`/reset-password` 可重置密码

## 今日视图 `/today`

- [ ] 显示今日待办与日记摘要
- [ ] 快速添加待办成功并出现在列表
- [ ] 快速写日记成功并出现在列表
- [ ] 勾选待办完成有视觉反馈
- [ ] 移动端布局可用（底部导航、表单可提交）

## 待办 `/todos`

- [ ] 创建、编辑、删除待办（删除有确认）
- [ ] 筛选（今日 / 全部 / 已完成）正常
- [ ] 设置截止日期、优先级、完成说明
- [ ] 详情页可查看与编辑

## 日记 `/diary`

- [ ] 创建、编辑、删除日记
- [ ] Markdown 正文预览正常
- [ ] 心情、标签、日期可保存

## 笔记 `/notes`

- [ ] 创建、编辑、删除笔记
- [ ] 分类筛选正常
- [ ] Markdown 渲染正常

## 计划 `/plans`

- [ ] 创建、编辑、删除计划
- [ ] 计划内创建/关联/取消关联待办
- [ ] 进度展示合理

## 复盘 `/reviews`

- [ ] 按日期创建与编辑复盘
- [ ] 关联日期的日记/待办上下文可见
- [ ] Markdown 内容保存与展示正常

## 长期目标 `/goals`

- [ ] 创建、编辑、删除目标
- [ ] 关联计划可选可保存
- [ ] 状态切换正常

## 搜索 `/search`

- [ ] 关键词搜索返回日记、待办、笔记等结果
- [ ] 按类型、标签、分类、日期筛选有效
- [ ] 结果可点击进入详情

## 设置 `/settings`

- [ ] 修改昵称、密码
- [ ] 标签增删改
- [ ] 分类增删改、拖拽排序
- [ ] 主题切换（明/暗）生效

## Docker 部署（可选，有 Docker 环境时）

- [ ] `docker compose up -d --build` 启动成功
- [ ] `docker compose exec app node create-user.js email pass "Name"` 可创建用户
- [ ] 重启容器后 `data/app.db` 数据仍在
- [ ] 拷贝 `data/app.db` 备份文件可恢复（复制回原路径后重启）

## 备注

- 本清单为手动验收，不包含自动化 E2E 测试
- 发现问题请记录：页面路径、操作步骤、期望与实际结果
